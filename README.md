Simple REST API in Play using Java Objects
------------------------------------------

#Dependencies

* Java 8
* Scala version 2.11.8
* SBT Version 0.13.11
* Play version 2.5.4

#Usage 

```shell
activator run
```
#Eclipse IDE
```
activator eclipse 
```
Scala Eclipse IDE 4.4.1 (if eclipse plugin does not set Java 8, set it up manually)

 http://scala-ide.org/
